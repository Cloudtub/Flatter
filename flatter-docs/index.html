Styles Styles customization and highlights

Stylesheets

Please note i have used break tags at some places to avoid mixing elements. They are just for theme purpose only. You should avoid in production.

Here's a list of the stylesheet files I'm using with this template, you can find more information opening each file:

File Name Description
bootstrap-override.css Bootstrap styles are changed from default in this.
layout.css This is layout file for basic theme usage and includes sidebar & header css.
library.css Libraries which are used in this theme are overridden here.
style.css This is main file and css for pages,background,widgets are in this file.
